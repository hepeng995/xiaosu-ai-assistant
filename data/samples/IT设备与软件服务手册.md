# IT 设备与软件服务手册

## 总则

本手册介绍公司提供的办公设备、软件服务与 IT 支持，帮助员工高效办公。IT 服务由 IT 部统筹，设备与账号实行"最小权限、按需开通、离职回收"原则。员工问"电脑怎么领""软件怎么装""VPN 怎么连""IT 找谁"时，应引用本手册。IT 服务请求统一在 OA《IT 服务台》提交，紧急情况拨打 IT 服务热线或 IM 联系 IT 服务号。

## 办公设备

### 电脑

入职时由公司配发办公电脑：

| 岗位类型 | 默认配置 |
|----------|----------|
| 研发/设计 | MacBook Pro 14 寸 或 高配 Windows 笔记本（16GB+ 内存） |
| 产品/运营 | MacBook Air 或 中配 Windows 笔记本（8GB+ 内存） |
| 行政/财务 | Windows 笔记本（8GB 内存） |
| 高管 | 按需配发 |

- 电脑归属公司，仅用于工作
- 首次开机由 IT 协助配置（域加入、软件安装、安全策略）
- 电脑故障联系 IT，不得自行拆机或重装系统
- 电脑更换周期：3-4 年（按公司设备更新计划）

### 外设

每个工位标配：

- 27 寸显示器 1 台（研发/设计可申请 2 台）
- 键盘、鼠标（可申请人体工学款）
- 显示器支架、笔记本支架

特殊外设（手绘板、机械键盘、专业耳机）需 OA 申请，主管审批。

### 手机与通讯

- 销售等岗位配发公司手机（含号码）
- 其他员工使用个人手机，享受通讯补贴（见《休假与福利政策》）
- 公司手机仅用于工作，离职归还

### 其他设备

- 测试机（研发）：按项目申请
- 投影/演示设备：会议室配备，前台可借
- 移动热点：出差借用，OA 申请

## 设备申请与领用

### 新员工领用

入职时由 IT 根据岗位自动配发电脑和外设，员工在《设备领用单》签字确认。

### 在职申请

- OA《IT 服务台》提交"设备申请"
- 注明需求、用途、预算
- 直属主管审批（5000 元以上需部门负责人审批）
- IT 采购、配置、交付

### 设备故障与更换

- 故障：联系 IT，工作时间内 2 小时响应
- 维修期间提供备用机（保证工作连续）
- 无法修复的走报废流程，配发新设备
- 设备更新周期：3-4 年，由 IT 统一规划

### 专业软件

特殊岗位需要的专业软件（如 CAD、SPSS、Matlab）需 OA 申请，IT 采购授权。

## 网络与 VPN

### 办公网络

- 有线：工位网口（千兆），自动获取 IP
- 无线：公司 Wi-Fi（WPA2 企业级，用工号密码登录）
- 访客 Wi-Fi：独立网络，仅供访客使用

### VPN

- 远程接入内网需使用 VPN
- 申请：OA《IT 服务台》提交，主管审批
- 客户端：公司统一 VPN 客户端，支持双因素认证
- 使用规范：

  - 不得将 VPN 账号、密码、令牌分享给他人
  - 使用后及时断开
  - 仅用于工作，不得用于私人用途
  - 设备需安装公司安全软件（杀毒、DLP）

### 远程办公

远程办公（居家、出差、异地）规范详见《远程办公与弹性工作管理办法》，IT 侧要点：

- 必须通过 VPN 接入公司内网
- 设备需安装安全软件
- 重要数据存公司云盘，不存本地
- 视频会议使用公司工具

## 信息安全

详细安全规范见《信息安全与账号权限》，IT 侧要点：

- **密码**：复杂密码（8 位以上，含大小写数字符号），90 天更换，不得复用
- **双因素**：邮箱、OA、代码仓库等关键系统强制开启双因素认证
- **杀毒**：公司电脑安装统一杀毒软件，不得卸载
- **加密**：笔记本电脑开启全盘加密（FileVault/BitLocker）
- **数据**：客户数据、员工信息、财务数据按分级保护，不得外传

## IT 支持

### 服务渠道

| 渠道 | 用途 | 响应 |
|------|------|------|
| OA《IT 服务台》 | 标准请求（设备、账号、软件） | 2 小时内响应，1-2 工作日解决 |
| IM IT 服务号 | 紧急问题、快速咨询 | 工作时间实时 |
| IT 服务热线 | 紧急故障（系统宕机、网络中断） | 实时 |
| 现场 | 设备故障、配置问题 | 工作时间上门 |

### 服务时间

- 工作日 9:00-18:00 全员支持
- 非工作时间：仅紧急故障响应（系统宕机、安全事件）
- 节假日：值班 IT 处理紧急

### 自助服务

- 密码重置：OA 自助门户或 IM 机器人
- 软件安装：公司软件中心自助下载（已授权软件）
- 知识库：IT 文档、FAQ、操作手册在知识库

## 设备维护与归还

### 日常维护

- 定期更新操作系统和安全补丁（IT 推送或自行更新）
- 定期备份重要数据（公司云盘）
- 保持设备清洁、安全（不安装来路不明软件）
- 发现异常（中毒、卡顿、丢失）立即联系 IT

### 设备归还

- 离职、调岗时归还所有公司设备
- IT 检查设备完好性、数据清除
- 遗失或人为损坏的按公司规定赔偿

## 附则

本手册由 IT 部负责维护，设备型号、软件版本、服务流程会随技术演进调整，以 IT 部通知为准。员工对 IT 服务有建议或投诉，可联系 IT 部（it@company.com）或主管。紧急安全事件（账号被盗、数据泄露）应立即报告 IT 和直属主管。
